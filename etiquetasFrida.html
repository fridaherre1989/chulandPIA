DOCTYPE HTML Es la que indica de que tipo de documento debe abrir el navegador
y que sintaxis y gramática utiliza su código.		 doctype html(Abrir el navegador del documento) Enlace del DOCTYPE HTML
HEAD Es la que provee información general (metadatos)
acerca del documento, incluyendo su titulo y enlaces a scripts y hojas de estilos.		 head(informacion general) Enlade del HEAD
BODY Es la que representa el contenido
de un documento HTML.		 body(contenido) Enlace de BODY
HTML Es la que define el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan
otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la
funcionalidad del (JavaScript). 		html(contenido de la web) Enlace de HTML
H1 A H6 Son las que se utilizan para definir los encabezados, siendo (h1) el mas importante y el (h6) el menos
importante. Cada etiqueta representa un nivel encabezado diferente, donde el tamaño y la importancia visual
disminuyen a medida que se incrementa el numero (de 1 a 6). 		h1 a h6(temas y subtemas) Enlace de H1 A H6
p Es el apropiado para distribuir el texto en párrafos. p(parrafo) Enlace de P
HR Es la que representa un cambio de tema entre parrafos (por ejemplo, un cambio de escena en una historia,
un cambio de tema en una sección). En versiones previas de HTML representa una linea horizontal.		 hr(linea horizontal) Enlace de HR
BR Es la que produce un salto de linea en el texto. br(salto de linea) Enlace de BR
UL Crea una lista no ordenada ul(viñetas) Enlace de UL
STRONG O B Es el apropiado para marcar con especial énfasis las partes más importantes de un texto. strong o b(letras negritas) Enlace de STRONG O B
i Muestra el texto marcado con un estilo en cursiva. i(cursiva) Enlace de i
u Muestra el texto subrayado. u(subrayar) Enlace de u
MARK Es la que representa un texto marcado o resaltado como referencia o anotación, debido a su relevancia o
importancia en un contexto particular.		 mark(marca texto) Enlace de MARK
SMALL Hace el tamaño del texto una talla más pequeña (por ejemplo, de largo a mediano, o de pequeño a extra pequeño)
que el tamaño minimo del fuente del navegador.		 small(letras pequeñas) Enlace de SMALL
EM Es el apropiado para marcar con énfasis las partes importantes de un texto. em(enfasis) Enlace de EM
CITE Marca una referencia a una fuente,

Q Indica que el texto adjunto

ABBR Es la que representa una abreviacion o acronimo;

CODE Es el apropiado para marcar el código de un programa. code(codigo) Enlace de CODE
SUB Define un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmete más pequeño,

SUP Define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño,
que el tramo principal del texto, es decir, en superindice.		 sup(superindice) Enlace de SUP
DEL Marca las partes de un texto o documento

TABLA DE ETIQUETAS
Etiqueta Explicación Ejemplo / Vista Referencia
DOCTYPE HTML Indica al navegador qué tipo de documento se está utilizando, así como las reglas de sintaxis que debe seguir para interpretarlo. <!DOCTYPE html> Referencia de DOCTYPE HTML
HEAD Sección que contiene datos generales del documento, como su título, enlaces a estilos, scripts o metadatos. <head>...</head> Referencia de HEAD
BODY Es la parte del documento HTML que reúne todo el contenido visible de la página. <body>...</body> Referencia de BODY
HTML Elemento raíz que marca la estructura y semántica del contenido en la web. Normalmente se combina con CSS para dar estilo y JavaScript para añadir funciones. <html>...</html> Referencia de HTML
H1 A H6 Definen títulos y subtítulos. El (h1) es el encabezado más importante y el (h6) el de menor jerarquía. <h1>Título</h1> ... <h6>Subtítulo</h6> Referencia de H1 a H6
P Sirve para escribir párrafos de texto. <p>Este es un párrafo.</p> Referencia de P
HR Genera una línea divisoria que indica un cambio de tema o separación de secciones. <hr> Referencia de HR
BR Produce un salto de línea en el texto. <br> Referencia de BR
UL Crea una lista con viñetas sin orden específico. <ul><li>Elemento</li></ul> Referencia de UL
STRONG / B Se usan para remarcar palabras o frases importantes mostrando el texto en negrita. <strong>Texto importante</strong> o <b>Texto importante</b> Referencia de STRONG y B
I Hace que el texto se muestre en estilo cursivo. <i>Texto en cursiva</i> Referencia de I
U Subraya el texto al que se aplica. <u>Texto subrayado</u> Referencia de U
MARK Resalta texto importante como si estuviera marcado con un resaltador. <mark>Texto resaltado</mark> Referencia de MARK
SMALL Reduce el tamaño de la tipografía respecto al texto normal. <small>Texto más pequeño</small> Referencia de SMALL
EM Da énfasis al texto, normalmente con estilo en cursiva. <em>Texto enfatizado</em> Referencia de EM
CITE Se usa para indicar el título de una obra o referencia a un autor. <cite>Nombre de la obra</cite> Referencia de CITE
Q Sirve para insertar citas cortas dentro de un párrafo. <q>Cita corta</q> Referencia de Q
BLOCKQUOTE Indica citas largas de texto de otro autor u obra. <blockquote>Cita extensa</blockquote> Referencia de BLOCKQUOTE
ABBR Se usa para escribir abreviaciones o siglas. Con el atributo (title) se puede especificar su significado. <abbr title="Ejemplo">Sigla</abbr> Referencia de ABBR
CODE Identifica fragmentos de código dentro del texto. <code>Código aquí</code> Referencia de CODE
SUB Muestra el texto en subíndice, es decir, más bajo y pequeño que el resto del texto. <sub>Texto subíndice</sub> Referencia de SUB
SUP Muestra el texto en superíndice, más alto y pequeño que el texto normal. <sup>Texto superíndice</sup> Referencia de SUP
DEL Marca texto que ha sido eliminado o reemplazado. <del>Texto eliminado</del> Referencia de DEL
IMG Inserta imágenes en un documento HTML. <img src="ruta" alt="Descripción"> Referencia de IMG
IFRAME Permite incrustar otra página HTML dentro de la actual. <iframe src="pagina.html"></iframe> Referencia de IFRAME
VIDEO Permite añadir archivos de video directamente en la página. <video src="video.mp4"></video> Referencia de VIDEO
